yepos programma vocabularium pro PalmOS est. insignia eius principia sunt:

  1. Libertas Stallmaniana: id uti redistribuique licet sub conditionibus GNU GPLv3+
  2. Programma de machinis veterioribus curat: in machinis cum PalmOS 2.0 atque 16 MHz DragonBall currit valde vehementer (verum vocabularios magnos portare machinae tantulae nequeant)
  3. Eius programmata in computatro maiore currentes in C scribuntur neque excidunt C89 (praeter zlib usus) singulosque scapos objectivos per programma habent portabilitatis amplioris et facile struendi gratia.
  4. Datorum basium non solum compilatrum, sed etiam decompilatrum habet; quod decompilatrum litteratim ullius datorum basis a compilatro creatae fontem regenerat
  5. Datorum bases compressas protegit
  6. Aspectus simplices atque mores graves; e.g. inventus linea nova excitatur; articuli superiore aut inferiore parte tabulis pulsibus labuntur; nihil fenestrarum titulorum, nihil clavium venustorum, nihil inutilium volutarum

yepos — программа-словарь для PalmOS. главные её особенности суть:

  1. Свобода в столмановском смысле: программу можно применять и распространять на условиях GNU GPLv3+
  2. Программа ориентирована на старые устройства: она довольно быстро идёт на устройствах с PalmOS 2.0 и процессорами DragonBall с тактовой частотой 16 МГц (правда, большие словари на таких машинках не умещаются)
  3. Утилиты для обработки баз данных пишутся на C в рамках C89 (не считая применения библиотеки zlib), одним объектным файлом на программу, так что их должно быть легко переносить и собирать
  4. В утилиты включаются как компилятор, так и декомпилятор баз данных, позволяющий в точности восстанавливать исходный текст любой базы данных, созданной компилятором
  5. Поддержка сжатых баз данных
  6. Безыскусный внешний вид и несуетливое поведение; напр., поиск запускается переводом строки, статьи перелистываются нажатием на верхнюю или нижнюю половину экрана; ни заголовков окон, ни изощрённых кнопок, ни бесполезных завитушек

yepos is another dictionary program for PalmOS. it's primary features are:

  1. RMS-sense freedom, you can use and distribute yepos under the terms of the GNU GPLv3+
  2. It is older devices-oriented: it runs on PalmOS 2.0 devices with 16 MHz DragonBall quite fluently (unfortunately, they can't bear any really large dictionary)
  3. It's host-side programs are written in C within C89 standard (apart from dependency on zlib), single object file per program, so should be very portable and easy to build
  4. it has both database compiler and decompiler to regenerate the database source file bit-to-bit
  5. compressed databases support
  6. minimalistic user interface, e.g. find action is triggered with carriage return stroke; articles are scrolled with tap on upper/lower half of the screen; no form titles, no fancy buttons, no other non-functional decorations
